Article

논문, 리뷰, 정리 글을 articles/ 폴더에 .md 파일로 올리면 이 페이지 왼쪽 메뉴에 자동으로 노출되도록 구성했습니다.

올리는 방법
  1. articles/ 폴더에 새 .md 파일을 추가합니다.
  2. 파일 첫 줄에 # 제목을 적으면 화면 제목으로 바로 사용됩니다.
  3. 파일명은 YYYY-MM-DD-title.md처럼 두면 정리하기 편합니다.
자동 처리
  • GitHub 저장소의 articles/ 폴더를 읽어 목록을 만듭니다.
  • md 파일은 페이지 안에서 바로 렌더링됩니다.
  • 원본 md 파일 열기 링크도 함께 제공합니다.
