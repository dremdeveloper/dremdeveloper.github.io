업로드 파일 안내
================

같은 폴더에 아래 4개 파일을 함께 올리면 됩니다.

1. boj-archive-search.html  -> 실제 검색 페이지
2. boj-archive.html         -> 검색 페이지로 즉시 리다이렉트
3. boj-archive.css          -> 공통 스타일
4. boj-archive-data.js      -> 문제 데이터

권장 연결 경로
- 사이트 메뉴에는 boj-archive-search.html 을 직접 연결하는 편이 가장 깔끔합니다.
- 기존 링크가 boj-archive.html 을 보고 있어도 자동으로 검색 페이지로 이동합니다.

이번 수정 반영 사항
- 상단 문제 검색 소개 프레임 삭제
- 필터는 통합 검색 / 티어 그룹 / 컬렉션 주제 / 알고리즘만 유지
- 빠른 컬렉션 프레임 삭제
- 빠른 알고리즘 프레임 삭제
- 태그 열 삭제
- 티어 색상을 티어 그룹별로 적용
- 카드형 대신 리스트형 결과 + 정답률/맞힌 사람 프로그래스바 적용
