문제 검색 페이지 배포 구조
=========================

현재 문제 검색 페이지는 루트의 `boj-archive-search.html`을 직접 배포합니다.
검색 데이터는 `boj_archive_site/boj-archive-data.js`에서 불러옵니다.

배포 시 필요한 파일
------------------
1. boj-archive-search.html         -> 실제 검색 페이지
2. boj_archive_site/boj-archive-data.js -> 검색 데이터
3. assets/styles.css               -> 블로그 공통 테마 스타일
4. assets/site.js                  -> 공통 상단 메뉴/푸터 스크립트

메모
----
- 사이트 메뉴에서는 `boj-archive-search.html`을 직접 연결합니다.
- 별도 전용 CSS 없이 블로그 메인 테마에 맞춘 공통 스타일을 사용합니다.
- 페이지에는 문제 검색 기능만 남기고 나머지 소개/부가 영역은 제거했습니다.
