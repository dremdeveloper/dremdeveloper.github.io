# 샘플 논문 아티클

- 날짜: 2026-03-21
- 분류: 생각정리
- 설명: `articles/` 아래 서브카테고리 폴더에 md 파일을 올리면 왼쪽 메뉴에 분류별로 자동 표시되는 예시 문서입니다.

## 이렇게 쓰면 됩니다

논문 요약, 리뷰, 구현 메모를 평소처럼 Markdown으로 작성하면 됩니다.

### 예시 섹션

1. 문제 정의
2. 핵심 아이디어
3. 실험 결과
4. 느낀 점

> GitHub 저장소의 `articles/` 폴더 아래에 카테고리별 하위 폴더를 유지하면 이 페이지가 자동으로 목록을 읽어옵니다.

```text
articles/
├── AI 논문/
│   └── 2026-03-22-another-paper-review.md
├── 트러블 슈팅/
├── 유용한 지식 및 팁/
└── 생각정리/
    └── 2026-03-21-sample-article.md
```

## 팁

- 첫 번째 `# 제목`은 화면 제목으로 사용됩니다.
- 파일명은 날짜를 붙이면 정렬하기 쉽습니다.
- 원본 md 링크도 함께 열 수 있습니다.
