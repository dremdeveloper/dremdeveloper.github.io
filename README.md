# Park Blog Site

GitHub Pages에 바로 올릴 수 있는 정적 사이트입니다.

## 구성
- `index.html`: 메인 소개, 경력, 학력, 보유기술, 도서
- `lectures.html`: 좌측 강의 목록 / 우측 플레이어 구조의 강의 페이지
- `study.html`: 좌측 자료 목록 / 우측 뷰어 구조의 공부자료 페이지
- `projects/python-flow-visualizer.html`: 파이썬 알고리즘 실행 흐름 시각화 자료

## 배포
이 폴더 전체를 GitHub Pages 저장소 루트에 올리면 됩니다.

## 메모
강의 데이터는 `assets/data.js`에 들어 있습니다. 공부자료를 추가할 때도 같은 파일에 항목만 추가하면 됩니다.
