# 적용 가이드

## 업로드할 파일
- `study-solutions.html`

이번 버전은 **문제 데이터, 스타일, 스크립트가 모두 한 파일에 들어 있는 단일 HTML**입니다.
별도의 `assets/` 폴더나 `data/problems.json` 없이도 바로 동작합니다.

## 메뉴 연결
기존 홈페이지에서 아래 링크만 연결하면 됩니다.

```html
<a href="/study-solutions.html" class="nav-link">문제 해설</a>
```

## 이번 수정 사항
- 상단의 큰 소개 프레임 삭제
- 검색창 / 분류 필터 삭제
- `문제를 선택해 주세요` 빈 프레임 삭제
- `책 기준 해설 범위` 삭제
- 상단 배지 영역 삭제
- 버튼 영역 정리
- `문제 분류`는 유지
- `힌트`와 `간단 해설`을 하나의 접기 패널로 통합
- 힌트/해설은 처음에는 닫힌 상태로 시작
- Python / C++ 전환 시 왼쪽 메뉴가 흔들리지 않도록 레이아웃 고정
- 86번~100번 문제 제외 유지

## 메모
- 대상 문제는 `programmers` 링크가 있는 문제들만 반영했습니다.
- 정답 코드는 GitHub raw 원본을 브라우저에서 직접 불러옵니다.
- 브라우저 네트워크 정책에 따라 코드 로드가 일시적으로 실패할 수 있습니다.
